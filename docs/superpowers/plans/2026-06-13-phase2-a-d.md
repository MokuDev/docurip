# Docurip Phase 2 — Persistenz, History, Cancel, Export

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Jobs über App-Neustart persistieren, History-View voll funktional machen, Crawl-Jobs abbrechen können, Ergebnisse als ZIP exportieren.

**Architecture:** Jobs werden als JSON auf Disk (`app_data_dir/jobs/<job_id>.json`) gespeichert. Der `AppState` lädt beim Start alle gespeicherten Jobs. Der Orchestrator speichert nach jedem Seiten-Update + am Ende des Jobs. Cancel verwendet den bestehenden `should_stop` AtomicBool. Export bündelt Output-Ordner via `zip` Crate.

**Tech Stack:** Rust (tauri, serde_json, tokio, zip), React (TypeScript), Tailwind CSS

```rust
use std::path::Path;

impl AppState {
    pub async fn save_job_to_disk(dir: &Path, job: &CrawlJob) -> anyhow::Result<()> {
        tokio::fs::create_dir_all(dir).await?;
        let path = dir.join(format!("{}.json", job.id));
        let json = serde_json::to_string_pretty(job)?;
        tokio::fs::write(path, json).await?;
        Ok(())
    }

    pub async fn load_job_from_disk(dir: &Path, job_id: &str) -> anyhow::Result<CrawlJob> {
        let path = dir.join(format!("{}.json", job_id));
        let json = tokio::fs::read_to_string(path).await?;
        let job = serde_json::from_str(&json)?;
        Ok(job)
    }

    pub async fn load_all_jobs(dir: &Path) -> anyhow::Result<Vec<CrawlJob>> {
        let mut jobs = Vec::new();
        if !dir.exists() {
            return Ok(jobs);
        }
        let mut entries = tokio::fs::read_dir(dir).await?;
        while let Some(entry) = entries.next_entry().await? {
            let path = entry.path();
            if path.extension().and_then(|s| s.to_str()) == Some("json") {
                let json = tokio::fs::read_to_string(&path).await?;
                if let Ok(job) = serde_json::from_str::<CrawlJob>(&json) {
                    jobs.push(job);
                }
            }
        }
        Ok(jobs)
    }

    pub async fn delete_job_from_disk(dir: &Path, job_id: &str) -> anyhow::Result<()> {
        let path = dir.join(format!("{}.json", job_id));
        if path.exists() {
            tokio::fs::remove_file(path).await?;
        }
        Ok(())
    }
}
```

```rust
use std::path::PathBuf;

pub struct AppState {
    pub active_jobs: RwLock<HashMap<String, JobHandle>>,
    pub persisted_jobs: RwLock<HashMap<String, CrawlJob>>,
    pub persist_dir: PathBuf,
}

impl Default for AppState {
    fn default() -> Self {
        Self {
            active_jobs: RwLock::new(HashMap::new()),
            persisted_jobs: RwLock::new(HashMap::new()),
            persist_dir: PathBuf::new(),
        }
    }
}

impl AppState {
    pub async fn init(persist_dir: PathBuf) -> Self {
        let mut state = Self {
            active_jobs: RwLock::new(HashMap::new()),
            persisted_jobs: RwLock::new(HashMap::new()),
            persist_dir,
        };
        if let Ok(jobs) = Self::load_all_jobs(&state.persist_dir).await {
            let mut map = state.persisted_jobs.write().await;
            for job in jobs {
                map.insert(job.id.clone(), job);
            }
        }
        state
    }

    pub async fn persist_job(&self, job: &CrawlJob) -> anyhow::Result<()> {
        Self::save_job_to_disk(&self.persist_dir, job).await?;
        let mut map = self.persisted_jobs.write().await;
        map.insert(job.id.clone(), job.clone());
        Ok(())
    }

    pub async fn remove_persisted_job(&self, job_id: &str) -> anyhow::Result<()> {
        Self::delete_job_from_disk(&self.persist_dir, job_id).await?;
        let mut map = self.persisted_jobs.write().await;
        map.remove(job_id);
        Ok(())
    }
}
```

**WICHTIG:** `AppState::default()` muss bleiben für `manage(AppState::default())` in `lib.rs` — aber wir ersetzen das durch `init()`.

```rust
#[tauri::command]
pub async fn delete_job(job_id: String, state: State<'_, AppState>) -> Result<(), String> {
    // Stop if active
    {
        let jobs = state.active_jobs.read().await;
        if let Some(handle) = jobs.get(&job_id) {
            handle.should_stop.store(true, Ordering::Relaxed);
        }
    }
    // Remove from active
    {
        let mut jobs = state.active_jobs.write().await;
        jobs.remove(&job_id);
    }
    // Remove from disk
    state.remove_persisted_job(&job_id).await.map_err(|e| e.to_string())?;
    Ok(())
}
```

```rust
#[tauri::command]
pub async fn list_jobs(state: State<'_, AppState>) -> Result<Vec<CrawlJob>, String> {
    let mut result = Vec::new();
    let mut seen = std::collections::HashSet::new();

    // Active jobs first (they have the latest state)
    {
        let jobs = state.active_jobs.read().await;
        for (_, handle) in jobs.iter() {
            let job = handle.job.read().await.clone();
            seen.insert(job.id.clone());
            result.push(job);
        }
    }

    // Then persisted jobs (skip if already active)
    {
        let jobs = state.persisted_jobs.read().await;
        for (_, job) in jobs.iter() {
            if !seen.contains(&job.id) {
                result.push(job.clone());
            }
        }
    }

    Ok(result)
}
```

```rust
use std::fs::File;
use std::io::{Read, Write};
use std::path::Path;
use zip::write::FileOptions;

pub fn zip_directory(src_dir: &Path, dst_file: &Path) -> anyhow::Result<()> {
    let file = File::create(dst_file)?;
    let mut zip = zip::ZipWriter::new(file);
    let options = FileOptions::default().compression_method(zip::CompressionMethod::Deflated);

    zip_directory_inner(src_dir, src_dir, &mut zip, &options)?;
    zip.finish()?;
    Ok(())
}

fn zip_directory_inner(
    base: &Path,
    current: &Path,
    zip: &mut zip::ZipWriter<File>,
    options: &FileOptions,
) -> anyhow::Result<()> {
    for entry in std::fs::read_dir(current)? {
        let entry = entry?;
        let path = entry.path();
        let name = path.strip_prefix(base)?.to_string_lossy();

        if path.is_file() {
            zip.start_file(name, *options)?;
            let mut f = File::open(&path)?;
            let mut buffer = Vec::new();
            f.read_to_end(&mut buffer)?;
            zip.write_all(&buffer)?;
        } else if path.is_dir() {
            zip.add_directory(name, *options)?;
            zip_directory_inner(base, &path, zip, options)?;
        }
    }
    Ok(())
}
```

- [ ] **Step 3: `export_job` Command**

```rust
#[tauri::command]
pub async fn export_job(
    job_id: String,
    state: State<'_, Arc<AppState>>,
    app: AppHandle,
) -> Result<String, String> {
    // Get job to find output dir
    let job = {
        let jobs = state.active_jobs.read().await;
        if let Some(handle) = jobs.get(&job_id) {
            handle.job.read().await.clone()
        } else {
            let jobs = state.persisted_jobs.read().await;
            jobs.get(&job_id).cloned().ok_or("Job not found")?
        }
    };

    let output_dir = std::path::PathBuf::from(&job.config.output_dir).join(&job_id);
    if !output_dir.exists() {
        return Err("Output directory not found".to_string());
    }

    // Ask user for save location
    let save_path = app.dialog().file().blocking_save_file();
    let save_path = save_path.ok_or("No file selected")?;
    let save_path = save_path.into_path().map_err(|_| "Invalid path")?;
    let save_path = if save_path.extension().is_none() {
        save_path.with_extension("zip")
    } else {
        save_path
    };

    crate::export::zip_directory(&output_dir, &save_path)
        .map_err(|e| e.to_string())?;

    Ok(save_path.to_string_lossy().to_string())
}
```
